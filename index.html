❓ FAQ

Common
Questions.

Is SPS Browser Pro really free? ▼
Yes — 100% free, always. SPS Browser Pro is open source and runs as a Python script. There are no subscriptions, no premium tiers that hide features, and no ads inside the browser.
Does it collect my data or send telemetry? ▼
No. SPS Browser Pro was independently engineered to respect your privacy. Zero telemetry, zero tracking, zero cloud sync without explicit user consent. Your data stays on your device.
What are the system requirements? ▼
Python 3.10 or newer, PyQt6, PyQt6-WebEngine, and optionally the requests library. Works on Windows 10+, Ubuntu 20.04+, macOS 12+. At least 512MB RAM recommended; 4GB+ for heavy tab use.
How does the ad blocker work? ▼
SPS uses a network-level request interceptor built directly into QtWebEngine. Requests to known ad, tracker, crypto mining, and social surveillance domains are blocked before any data is downloaded — not filtered after — giving both privacy and speed benefits.
Can I import my Chrome/Firefox bookmarks? ▼
Yes! Use Menu → Import Bookmarks to import HTML bookmark exports from Chrome, Firefox, Edge, or any Netscape-format bookmark file. JSON import is also supported.
What is the Command Palette? ▼
Press Ctrl+K to open a universal search interface that lets you instantly switch tabs, open bookmarks, visit history entries, or trigger any browser action — all without touching the mouse. It fuzzy-matches everything in real time.
Is there a mobile version? ▼
Not yet — SPS Browser Pro is currently desktop-only (Windows, Linux, macOS). A mobile-optimized version and an Android app are on the roadmap. Watch our update page for announcements.
How do I update to a new version? ▼
If Auto-update is enabled in Settings, SPS Browser checks the update server on startup. When a new version is available, it shows an update dialog, downloads the installer, runs it silently, and removes the old version automatically.
