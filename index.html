Your Privacy Comes First

We don't track you, sell your data, or show you ads. SPS Browser Pro is built from the ground up with privacy as the default, not an afterthought.

🚫 No Telemetry

SPS Browser Pro collects zero browsing data. No usage statistics, no crash reports sent to servers, no analytics — nothing leaves your device.

🍪 Cookie Isolation

Total Cookie Protection ensures cookies are fully isolated to each website. Cross-site tracking through cookies is structurally impossible.

🔒 HTTPS-Only Mode

Enable HTTPS-Only mode to automatically upgrade all connections to encrypted HTTPS. Get a warning before visiting any plain HTTP site.

🕵️ Incognito Tabs

Open incognito tabs with Ctrl+Shift+N. Browsing history, cookies, and form data are never saved. Session disappears on close.

🖐 Fingerprint Protection

Advanced canvas and WebGL fingerprint protection makes it harder for websites to uniquely identify your browser across sessions.

🧹 Clear on Exit

Enable "Clear data on exit" to automatically wipe cookies, cache, and site data every time you close the browser — zero traces left.

📡 Do Not Track

DNT:1 header is sent with every request, signaling to websites that you do not consent to behavioral tracking.

⛏ Crypto Protection

Cryptomining scripts from 12+ known domains (Coinhive, CryptoLoot, etc.) are blocked at the request level before consuming your CPU.

📵 Social Blocker

Facebook, Twitter, LinkedIn, and Pinterest tracking pixels are blocked — you can use social sites without being tracked across the web.

Frequently Asked Questions

Everything you need to know about SPS Browser Pro.

Is SPS Browser Pro free?›
Yes, 100% free. There are no paid plans, no premium features, no subscriptions, and no ads inside the browser. SPS Browser Pro is free forever.
Does it collect my browsing data?›
Absolutely not. SPS Browser Pro stores all data locally on your device. No telemetry, no analytics, no crash reports, and no usage data is ever sent to any server. Your browsing stays entirely private.
What OS does it support?›
SPS Browser Pro runs on Windows 10/11, Linux (Ubuntu, Debian, Fedora), and anywhere Python 3.10+ with PyQt6 is available. macOS is supported when running from source.
How do I install it on Linux?›
Install the dependencies with pip install PyQt6 PyQt6-WebEngine requests then run python sps_browser.py. The Linux .sh script automates this process.
Can I use Chrome extensions?›
SPS Browser Pro uses QtWebEngine which doesn't support the Chrome extension store. However, it has a built-in Userscript Manager where you can inject custom JavaScript and CSS into any page — similar to Greasemonkey/Tampermonkey.
How does the password manager work?›
Passwords are stored locally in your profile folder using SHA-256 hashing with a random salt. They never leave your device. You can export them to CSV if needed.
What are @ shortcuts in the URL bar?›
Type @history, @bookmarks, @downloads, @settings, @passwords, @notes, @reader, @screenshot, @pdf, @devtools, @cookies, @extensions, @focus, @source, @print or @translate directly into the address bar to instantly open those features.
How do I update SPS Browser Pro?›
Click the ⋮ menu → "Check for Updates". If an update is available, SPS Browser Pro will download the new installer automatically and launch it. The browser self-updates. You can also enable "Auto-update" in Settings → Notifications.
Can I have multiple user profiles?›
Yes. SPS Browser Pro has full multi-profile support. Each profile has its own settings, bookmarks, history, passwords, cookies, and session data — completely isolated from other profiles.
What is Focus Mode?›
Focus Mode (F11 or Ctrl+Shift+F) hides the toolbar, sidebar, bookmarks bar, and status bar — leaving only the web content visible for distraction-free reading and work. Press F11 again to exit.
Is there a mobile version?›
SPS Browser Pro is currently a desktop application for Windows, Linux, and macOS. A mobile version is under consideration for future development.

Getting Started

After installing SPS Browser Pro, you'll be greeted by the Profile Selector. Create a profile or use the Default one to begin. Your profile stores all settings, bookmarks, history, passwords, and extensions.

First Launch Checklist

On first launch, consider configuring: your preferred search engine (Settings → Search), download folder (Settings → Downloads), and privacy features (Settings → Privacy).

💡 Tip: Press Ctrl+L at any time to instantly focus the URL bar. Start typing to search or navigate.

Tab Management

SPS Browser Pro supports unlimited tabs with full management features. Tabs can be pinned, muted, moved to new windows, and reopened after accidental closure.

Tab Actions

Right-click any tab to: Duplicate, Pin, Mute, Move to New Window, Add to Reading List, Close, Close Others, or Close Tabs to Right.

💡 Tip: Closed tabs are remembered. Press Ctrl+Shift+T to reopen the last closed tab.

Privacy Features

All privacy features are active by default. Open Settings → Privacy & Security to fine-tune which protections are enabled.

Privacy Shield

The built-in interceptor blocks ads, trackers, cryptominers, and social pixels at the network level — before any request leaves your machine.

Incognito Mode

Press Ctrl+Shift+N to open an incognito tab. History, cookies, and form data are never saved. Each incognito session is fully isolated.

Password Manager

Access the password manager from the sidebar (🔐) or via @passwords in the URL bar. Add entries manually or let auto-fill handle it on login forms.

Generating Passwords

Click ⚡ Generate in the Password Manager to create a secure password. Adjust length (8–64) and toggle symbols. Click ✓ Use to apply it to the password field.

Settings

Open Settings via the sidebar (⚙️), ⋮ menu → Settings, or by typing @settings. There are 14 sections covering every browser behaviour.

Key Settings Sections

General — startup, zoom, homepage, tab behaviour
Privacy & Security — all privacy toggles
Ad & Tracker Blocking — individual blocking engines
Appearance — themes, font size, focus mode
Network & Proxy — custom proxy, user agent spoofing
Performance — hardware acceleration, tab memory limits

Themes

Go to Settings → Appearance to choose from 8 preset themes or create your own. Changes apply live across the entire UI including tabs, sidebar, toolbar, and dialogs.

🎨 Themes available: ☀️ Light · 🌙 Dark · 🌊 Ocean · 🌲 Forest · 🌸 Pink · 🖤 Midnight · 🍊 Orange · 🟣 Purple

Profiles

Each profile is a completely separate browser identity with its own settings, bookmarks, history, cookies, passwords, notes, and extensions. Profiles are stored in ~/.sps_browser_v21/profiles/

Open Profile Manager from the sidebar (👤) or ⋮ menu → Manage Profiles. You can create, rename, or delete profiles — the Default profile cannot be deleted.

Auto-Updates

SPS Browser Pro checks for updates at sites.google.com/view/spsbrowserpro/updates. When a newer version is detected, the browser downloads the new installer and runs it automatically, then removes the old version.

Enable or disable auto-update in Settings → Notifications. Manual check: ⋮ menu → Check for Updates.
